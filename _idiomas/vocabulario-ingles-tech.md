## Desarrollo de Software

### Términos Básicos

| Inglés | Español | Ejemplo |
|--------|---------|---------|
| bug | error/fallo | "There's a bug in the code" |
| feature | funcionalidad | "We need to add this feature" |
| deploy | desplegar | "Let's deploy to production" |
| refactor | refactorizar | "This code needs refactoring" |
| debug | depurar | "I'm debugging the issue" |

### Frases Comunes en Reuniones

**Stand-up meetings:**
- "What did you work on yesterday?"
- "What are you working on today?"
- "Do you have any blockers?"

**Code review:**
- "Can you review my PR?" (Pull Request)
- "I left some comments on your code"
- "LGTM" (Looks Good To Me)

## Comunicación en Equipo

### Slack/Chat

| Frase | Significado |
|-------|-------------|
| "Heads up" | Aviso/advertencia |
| "FYI" | For Your Information |
| "ASAP" | As Soon As Possible |
| "EOD" | End Of Day |
| "OOO" | Out Of Office |

### Emails

**Iniciar:**
- "I hope this email finds you well"
- "Following up on our conversation..."
- "Just wanted to touch base about..."

**Cerrar:**
- "Let me know if you have any questions"
- "Looking forward to your response"
- "Best regards" / "Cheers"

## Entrevistas Técnicas

### Preguntas Comunes

- "Tell me about yourself"
- "Walk me through your experience"
- "Why do you want to work here?"
- "Describe a challenging project"
- "How do you handle tight deadlines?"

### Respondiendo

- "In my previous role, I..."
- "I would approach this by..."
- "One example that comes to mind is..."

## Documentación

### Escribir Documentación Técnica

```markdown
## Overview
Brief description of what this does.

## Installation
Steps to install.

## Usage
How to use it with examples.

## Contributing
How others can contribute.
```
